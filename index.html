Click the "Download File" button to download the file specified by Server URL + fileName to this device. Server URL should be server directory ending with /.

Click "Upload File to upload from the android to the server. The file name must exist in the RideCheck directory. The file name will be written to the specified directory on the server.

Folder "RideCheck" will be created on download (if it doesn't exist) in the default persistent file location, and the downloaded file will be written there. It will cause an error if the uploaded file does not exist.

This is a test of the Phonegap file transfer plugin
